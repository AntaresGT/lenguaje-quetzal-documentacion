## ¿Por qué Quetzal?

<CardGrid stagger>
	<Card title="Sintaxis en español" icon="rocket">
		Programar en tu idioma nativo hace que el código sea más fácil de leer y entender.
	</Card>
	<Card title="Tipado fuerte" icon="setting">
		Sistema de tipos robusto que previene errores comunes en tiempo de compilación.
	</Card>
	<Card title="Inmutabilidad por defecto" icon="approve-check">
		Las variables son constantes a menos que se especifique lo contrario, promoviendo código más seguro.
	</Card>
	<Card title="Facilidad de uso" icon="open-book">
		Sintaxis clara y expresiva diseñada para ser intuitiva para hispanohablantes.
	</Card>
</CardGrid>

## Ejemplo rápido

```qz
// Un programa simple en Quetzal
cadena nombre = "Mundo"
entero edad = 25
bool es_activo = verdadero

// Función para saludar
fn saludar(cadena persona) -> cadena {
    retornar "¡Hola, " + persona + "!"
}

// Programa principal
imprimir(saludar(nombre))
```
