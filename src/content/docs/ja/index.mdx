## 📚 コード例

Quetzalの力と優雅さを示すいくつかの例：

### データ構造
```javascript
// リストの操作
lista números = [1, 2, 3, 4, 5]
lista pares = números.filtrar(n => n % 2 == 0)
imprimir("偶数: " + pares.cadena())

// オブジェクトとプロパティ
objeto persona = {
    nombre: "田中太郎",
    edad: 30,
    ciudad: "東京"
}

imprimir("こんにちは " + persona.nombre)
```

### 関数 & 制御フロー
```javascript
// パラメータ付き関数
cadena saludar(cadena nombre) {
    devolver "こんにちは、" + nombre + "！"
}

// 条件分岐
si (edad >= 18) {
    imprimir("成人です")
} sino {
    imprimir("未成年です")
}

// ループ
para i desde 1 hasta 10 {
    imprimir("番号: " + i.cadena())
}
```

### 非同期操作
```javascript
// 非同期関数
asincrono promesa<cadena> obtener_datos() {
    aguardar tiempo_espera(1000)
    devolver "データの読み込み完了"
}

// awaitでの使用
cadena resultado = aguardar obtener_datos()
imprimir(resultado)
```

## 🎯 Quetzalを選ぶ理由

- **自然**: 読みやすく理解しやすいスペイン語の構文
- **強力**: 型安全性と高性能を備えた関数型パラダイム
- **軽量**: わずか56KB、あらゆるプロジェクトに適している
- **本番対応**: 現在Windows用、マルチプラットフォーム対応予定
- **コミュニティ**: アクティブなサポートと完全なドキュメント
