## 📚 Contoh Kode

Berikut adalah beberapa contoh untuk menunjukkan kekuatan dan keeleganan Quetzal:

### Struktur Data
```javascript
// Bekerja dengan daftar
lista números = [1, 2, 3, 4, 5]
lista pares = números.filtrar(n => n % 2 == 0)
imprimir("Bilangan genap: " + pares.cadena())

// Objek dan properti
objeto persona = {
    nombre: "Ana García",
    edad: 30,
    ciudad: "Madrid"
}

imprimir("Halo " + persona.nombre)
```

### Fungsi & Kontrol Alur
```javascript
// Fungsi dengan parameter
cadena saludar(cadena nombre) {
    devolver "Halo, " + nombre + "!"
}

// Kondisional
si (edad >= 18) {
    imprimir("Anda sudah dewasa")
} sino {
    imprimir("Anda masih di bawah umur")
}

// Loop
para i desde 1 hasta 10 {
    imprimir("Nomor: " + i.cadena())
}
```

### Operasi Asinkron
```javascript
// Fungsi asinkron
asincrono promesa<cadena> obtener_datos() {
    aguardar tiempo_espera(1000)
    devolver "Data berhasil dimuat"
}

// Penggunaan dengan await
cadena resultado = aguardar obtener_datos()
imprimir(resultado)
```

## 🎯 Mengapa Memilih Quetzal?

- **Alami**: Sintaks dalam bahasa Spanyol yang mudah dibaca dan dipahami
- **Kuat**: Paradigma fungsional dengan type safety dan performa tinggi
- **Ringan**: Hanya 56KB, cocok untuk proyek apapun
- **Siap Produksi**: Saat ini untuk Windows, dengan rencana multiplatform
- **Komunitas**: Dukungan aktif dan dokumentasi lengkap
