## üéØ Why Quetzal?

Quetzal is more than a programming language; it's a development philosophy that prioritizes:

- **Clarity**: Spanish syntax that facilitates understanding
- **Efficiency**: Optimized performance thanks to Rust
- **Simplicity**: Minimalist but powerful design
- **Functionality**: Functional paradigm for cleaner code
- **Accessibility**: Integrated tools for all levels
